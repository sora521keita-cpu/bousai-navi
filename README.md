# わが家の防災ナビ

「わが家の防災ナビ」は、自治体別の防災情報、緊急時の行動カード、防災ライブラリ、家族・職場向け共有メモをまとめて確認するためのWeb/PWA向けMVPです。

このアプリは、災害通知アプリや緊急地震速報アプリではありません。主目的は、平時に自治体・公的情報への導線を整理し、緊急時に外部リンクを開かなくても最低限の行動手順を確認できるようにすることです。

## MVPでできること

- 下部3タブで主要機能を切り替え
  - 地域マップ
  - 防災ライブラリ
  - 緊急
- 緊急タブへ常時アクセスできる固定ボタンを表示
- 東京都・埼玉県・千葉県・神奈川県のサンプル自治体を選択
- 自治体詳細で公式リンク・防災関連リンク枠を確認
- 自宅・職場・実家・よく行く場所として自治体を登録
- 登録エリアを `localStorage` に保存
- 仮リンクを仮リンクとして明示
- 防災ライブラリをカテゴリ・ユーザー属性で絞り込み
- 信頼度A/B/C/Dラベルを表示
- 必須リンクとして以下2件を表示
  - 東京備蓄ナビ「自分に合った備蓄を調べてみよう」
  - 東京都防災ホームページ「東京くらし防災」・「東京防災」の閲覧
- 緊急時の状況別行動カードを表示
- 行動カード内で手順・注意文・次の行動を確認
- Service Workerにより、最低限のアプリ本体・緊急行動カードをオフラインでも確認
- オフライン時に外部リンクや共有APIが使えない可能性を表示
- 家族向け・職場向け共有テキストを生成
- Web Share API、クリップボードコピー、手動コピー用テキストエリアに対応

## セットアップ方法

> 現在のMVPは外部APIキーを必要としません。

```bash
npm install
```

開発サーバーを起動する場合:

```bash
npm run dev
```

ビルド済みファイルを確認する場合:

```bash
npm run preview
```

## `npm run build` の実行方法

以下を実行すると、TypeScriptチェックと静的配信用の `dist/` 生成を行います。

```bash
npm run build
```

現在の `build` スクリプトは以下を順に実行します。

1. `tsc -b`
2. `node scripts/clean-build.mjs`
3. `tsc -p tsconfig.emit.json`
4. `node scripts/build.mjs`

`dist/` は生成物のためGit管理対象外です。

## 防災情報に関する注意書き

- このアプリは災害予測や安全判定を行いません。
- 「この避難所に行けば安全です」「この地域は安全です」といった断定はしません。
- 緊急時は、周囲の状況に応じて安全を最優先してください。
- 避難情報、津波情報、気象情報、ライフライン情報は、自治体・気象庁・消防・警察・インフラ事業者などの最新情報を確認してください。
- アプリ内の行動カードは、通信できない場合にも最低限の行動を確認するための補助です。
- 外部リンクは通信状況や提供元の変更により開けない場合があります。
- 登録エリアや共有文には、氏名・電話番号・詳細住所などの個人情報を保存しない方針です。
